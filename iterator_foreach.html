The .forEach() Method

.forEach() executes the same code for each element of an array.

See illustraton below: 

                const colors = ['Red', 'Blue', 'Green', 'Yellow'];
                colors.forEach(function(myColors){
                    console.log('I love ' + myColors);
                })

Output:

            I love Red
            I love Blue
            I love Green
            I love Yellow

The above code will log nicely formatted list of colors to the console as shown in the output section. Letâ€™s explore the syntax of invoking .forEach().

Syntax:

arrayName.forEach((element) => { // do something with element }); In this case, we have a variable called "colors", which is an array containing four elements. We are using the .ForEach() method on that array and passing it a function (arrow function). This arrow function takes one parameter - "element".
Inside the curly braces, you can perform any operation on that particular element.
Here, it's just printing out the color using the console.log() method.
