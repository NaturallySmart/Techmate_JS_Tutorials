Looping Through Objects

In JavaScript, you can use a loop to iterate over the properties of an object and perform operations using a for...in statement.
Loops are programming tools that repeat a block of code until a condition is met.
We have previously learnt how to loop through arrays.
For...in will execute a given block of code for each property in an object. 

    let business = {
        team: {
            ceo: {
                name: 'Ray',
                skills: 'FullStack Development',
                getstarted() { console.log('Lets start this project!')} 
            },
            
            'frontend Engineer': {
                name: 'John',
                skills:'React',
                design() {console.log ('I will get all the designs sorted')}
            },

            'Backend Engineer': {
                name: 'Tom',
                skills: 'NodeJS',
                deploy(){console.log('Deployment complete!')},
            },

            'Project Manager': {
                name: 'Chris',
                skills: "Agile/Scrum",
                plan() {console.log("Let's make a plan.")}
            },

            'Product Manager': {
                name: 'Jenny',
                skills: "User Research, UI/UX",
                research() {console.log('Gathering user feedback..')}
            }
        }
    };

    // using for in loop

    for (let teamMembers in business.team)
    {
        console.log(`${business.team[teamMembers].name} is the ${
            teamMembers} and has skills in 
            ${business.team[teamMembers].skills}`);
    }

In the above code,
1. We have created an object named `business` which contains another object called `team`. This nested object represents our team members along with their details like names and skills.
2. We used the for in loop to Iterate over each member of the `team` object using the variable `teamMembers`. Inside this loop, we can access properties of `business.team[teamMembers]` such as `business.team[teamMembers].name`, `business.team[teamMembers].skills`, etc.
