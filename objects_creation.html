Creating Object Literals

In JavaScript, we can create an object literal by assigning a value to the key of that property using curly braces {}. For Instance: 

                let person = { }; //person is an empty object
We fill an object with unordered data. This data is organized into key-value pairs. A key is like a variable name that points to a location in memory that holds a value.

A key’s value can be of any data type in the language e.g boolean, string, numbers etc, including functions or other objects.

Here's how you do it:
  • Create an object.
  • Add properties and values to the object.
  • Access the properties through dot notation (.) or square brackets ([])

How to Make a Key-value pair

To make a key-value pair, follow these steps:

  1. Start with an opening curly brace ({).
  2. Write your desired key (variable) followed by a colon (:).
  3. Then write the corresponding value for each key.
  4. Finally, end the object with a closing curly brace (}).
We make a key-value pair by writing the key’s name, or identifier, followed by a colon and then the value. We separate each key-value pair in an object literal with a comma (,). Keys are strings, but when we have a key that does not have any special characters in it, JavaScript allows us to omit the quotation marks:

For Example: 
                const car = {'Year Make': 2023, color: 'White'};
Frpm the above,

  • Car is the Object
  • 'Year Make' is a key that holds a value which is a number 2023. Quoatation marks are used here because it has a space character.
  • color is another key that holds a string value White. No quotes needed as there is no spaces or an special character.
