JavaScript Comparison and Logical Operators

Comparison and Logical operators are used to test for true or false.

Comparison Operators

When writing conditional statements, sometimes we need to use different types of operators to compare values. These operators are called comparison operators.

Below is a list of some handy comparison operators and their syntax:

  • Less than: <
  • Greater than: >
  • Less than or equal to: <=
  • Greater than or equal to: >=
  • Is equal to: ===
  • Is not equal to: !==


    • Comparison operators compare the value on the left with the value on the right.

    It can be helpful to think of comparison statements as questions. When the answer is “yes”, the statement evaluates to true, and when the answer is “no”, the statement evaluates to false.

    Comparison statement can be used for both numbers and strings.

    Logical operators are used to determine the logic between variables or values. They are often used in conditional statements to add another layer of logic to our code.

    The && Operator (Logical AND)

    The && operator returns true if both expressions are true, otherwise it returns false.

    The || Operator (Logical OR)

    The || operator returns true if one out of the two expressions are true, otherwise it returns false.

    The ! Operator (Logical NOT)

    The ! operator otehrwise known as the bang operator takes a true value and return it as false or it will take a false value and return it as true.
