The .findIndex() Method

The findIndex() method returns the index of the first element in an array that satisfies a certain condition (provided as a function). If no elements satisfy the condition, it will return -1. This is useful when you want to find the position of an item within an array.

Let's see an illustration: 

                const numbers = [4, 2, 7, 5, 8];
                const result = numbers.findIndex(num => {
                    return num > 6
                    });
                console.log(result); //Outputs: 2

                //To check what element has index of 2:

                console.log(numbers[2]) //Outputs: 7
The above illustration will return 2 because the number at position 2 which is 7, is greater than 6 and this is the first occurrence. Now let's check another illustration where the elements in the array does not meet the condition. 
                const numbs = [4, 2, 7, 5, 8];
                const newResult = numbs.findIndex(num => {
                    return num > 10;
                });
                console.log(newResult) // Outputs -1
