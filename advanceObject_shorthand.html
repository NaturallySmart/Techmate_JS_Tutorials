Advance Objects | Property Value Shorthand

In JavaScript, you can use the shorthand syntax to assign multiple properties at once in an object literal. This is known as property value shorthand and it can be referred to as destructuring. It was introduced in ES6.

In our previous lessons, we created a Factory Function that helped us create objects. We had to assign each property a key and value even though the key name was the same as the parameter name we assigned to it. Here's a reminder: 

                const carFactory = (year, horsepower, trim, name) => {
                    return {
                        year: year,
                        horsepower: horsepower,
                        trim: trim
                    }                
                };
Imagine if we had to include more properties, that process would quickly become tedious! But we can use a destructuring technique, called property value shorthand, to save ourselves some keystrokes. The example below works exactly like the example above: 
                const carFactory = (year, horsepower, trim, name) => {
                    return {
                        year,
                        horsepower,
                        trim
                    }                
                };
