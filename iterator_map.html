The .map() Method

The .map( ) method works similar like the .forEach() method. The only difference is that it returns a new array.

When .map() is called on an array, it takes an argument of a callback function and returns a new array! Take a look at an illustration of calling .map(): 

                const numbs = [1, 2, 3, 4, 5];
                const newNumbs = numbs.map(number => {
                    return number * 5;
                    });
                console.log(newNumbs);

Output:

            [5, 10, 15, 20, 25]

In the illustration above:

  • numbs is an array of numbers.
  • newNumbs will store the return value of calling .map() on numbs.
  • numbs.map will iterate through each element in the numbs array and pass the element into the callback function.
  • return number * 5 is the code we wish to execute upon each element in the array. This will save each value from the numbs array, multiplied by 10, into a new array called newNumbs.
