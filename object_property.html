Object Property Assignment

In JavaScript, objects are collections of properties and each property has a key (name) associated with a value. Objects are mutable meaning we can update them after we create them!

We can use either dot notation, ., or bracket notation, [], and the assignment operator, = to add new key-value pairs to an object or change an existing property.

See the Illustration below: 

                Car {'Fuel Type'} = 'Petrol';
                car.make = 'Toyota';
In the above Illustration:
  • The first line is an example of Object Property assignment where we assign value to Fuel type as Petrol using the bracket notation.
  • The second line is an example of Object Property assignment where we assign value to make as Toyota using the dot notation.
    • One of two things can happen with property assignment:
    • If the object does not have that property then it will be created for you.
    • If the object already contains that property then its existing value will be replaced by new one.

    Note: It’s important to know that although we can’t reassign an object declared with const, we can still mutate it, meaning we can add new properties and change the properties that are there.

    Illustration

                const car = {make: 'Toyota'};
            car = {make: 'Toyota'}; // TypeError: Assignment to constant variable.
            car.make = 'Honda'; // Changes the value of the make property.

            car.color = 'Red'; // Create a new key of 'color' with a value of 'red'

    You can delete a property from an object with the delete operator.

                const car = {
                make: 'Toyota';
                color: 'red';
                'Fuel Type': 'Petrol';
                owner: 'Ray Jones';
            }

            delete car.color; // removes the color property
