Function Declarations

To be able to use a function, you must first declare the function. There are several ways to declare a function, and one way to do so is using a function declaration. A function declaration has the function keyword, a name or identifier binded with the function just as in variables, followed by a parenthesis ( ), then the function body inside a cury brace { }.

syntax:

function functionName( ) {
// code statements i.e the function body;
}
