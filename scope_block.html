Block Scope

We have effectively be using block of codes in our previous lessons. We used this in functions as we as in our if statement lessons. A block is the code found inside a set of curly braces {}. Blocks help us group one or more statements together and serve as an important structural marker for organizing our code. In JavaScript, blocks are used to create scope.

A block of code could be a function, like this: 

            const logSkyColor = () => {
                let color = 'blue';
                console.log(color); // blue
            }
In the above, the function body is a block of Code.

A block of code is also used in the if else statement below.
