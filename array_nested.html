Nested Arrays

Arrays are capable of storing other arrays. Nested arrays are arrays that contains other arrays.
See a nested array below: 

                const nestedArr = [['Ray', 2], ['Julia', 'John']];
To access the nested arrays we can use bracket notation with the index value, just like we did to access any other element: 
                const nestedArr = [['Ray', 2], ['Julia', 'John']];

                console.log(nestedArr[1]); // Outputs ['Julia', 'John']
Notice that nestedArr[1] will grab the element in index 1 which is the array ['Julia', 'John']. Then, if we want to access the elements within the nested array we can chain, or add on, more bracket notation with index values. 
                const nestedArr = [['Ray', 2], ['Julia', 'John']];
                console.log(nestedArr[1][0]); // Outputs 'Julia'
                console.log(nestedArr[1][1]); // Outputs 'John'
In the above example, in the first console.log, [1] was used to access the array in index 1 which is ['Julia', 'John'], then the [0] was used to access the element in index 0 which is 'Julia'
