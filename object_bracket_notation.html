Accessing Object Properties

Bracket Notation

In JavaScript, you can also access the properties of an object using bracket notation. This is done by specifying the property's key within square brackets following the object variable.

Remember we have used brackets when finding the index of an array, e.g: 

                const arr = [A,6,7];
                console.log(arr[1]); // Outputs: 6
To use bracket notation to access an objectâ€™s property, we pass in the property name (key) as a string.

We must use bracket notation when accessing keys that have numbers, spaces, or special characters in them. Without bracket notation in these situations, our code would throw an error.

Illustration:

            const person = {
                'First Name': "John",
                'Last Name': "Doe",
                age: 35,
                'Active Status': true
            };

            //Using bracket Notation to access the keys

            person['First Name'];  //output: John
            person['Last Name'];   //output: Doe
            person['age'];         //output: 35
            person['Active Status'];//output: true
            person['Origin City'];  //output: undefined

With bracket notation you can also use a variable inside the brackets to select the keys of an object. This can be especially helpful when working with functions. 

                let getPersonProp(objectName, propName) => 
                person[propName];

                return getPersonProp(person, 'age'); // return 35
If we tried to write our getPersonProp() function with dot notation (objectName.propName) the computer would look for a key of 'propName' on our object and not the value of the propName parameter.
