The For Loop

Instead of writing out the same code over and over, loops allow us to tell computers to repeat a given block of code on its own. One way to give computers these instructions is with a for loop.

The typical for loop includes an iterator variable that usually appears in all three expressions. The iterator variable is initialized, checked against the stopping condition, and assigned a new value on each loop iteration. Iterator variables can have any name, but it’s best practice to use a descriptive variable name.

A for loop contains three expressions separated by a semi-colon (;) inside the parentheses:

  1. Initialization - This expression sets up the starting point or state of the loop. It's executed once before the loop begins and can also be used to declare the iterator variable.
  2. Condition - This expression determines whether the loop should continue executing. If the condition evaluates as true, then the loop continues; if it evaluates to false, the loop ends.
  3. an iteration statement is used to update the iterator variable on each loop.
The for loop syntax looks like this: 
              
                for (let counter = 0; counter < 5; counter++) {
                    console.log(counter);
                }
                
                In the above example, the output will be:
                    0
                    1
                    2
                    3
                    4
Example explained:
  • We start by declaring our iterator variable "counter" and setting it equal to zero (let counter = 0).
  • Next we set the condition for when the loop should stop. Our condition checks if the current value of "counter" is less than five (counter < 5).
  • Finally, we increment the value of "counter" by one with each loop iteration using the post-increment operator ++ (counter++). The result of the expression after the semicolon is returned, so in this case, it returns the original value of "counter".
  • The code block is inside of the curly braces, console.log(counter), will execute until the condition evaluates to false. The condition will be false when counter is greater than or equal to 5 — the point that the condition becomes false is sometimes called the stop condition.
This for loop makes it possible to write 0, 1, 2, and 3 programmatically.

Looping in Reverse

In the above example, the loop runs from 11 to 15. How about if we want it to count backward from 15 down to 11?

To run a backward for loop, we must:

  1. Set the iterator variable to the highest desired value in the initialization expression. (let counter = 15)
  2. Specify the condition for ending the loop as an inequality test against the lowest desired value. (counter >= 11)
  3. Use a decrement operator -- instead of an increment operator ++. (--counter)
