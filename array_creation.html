Creating an Array

One way we can create an array is to use an array literal. An array literal creates an array by wrapping items in square brackets [ ]. Remember from the previous exercise, arrays can store any data type — we can have an array that holds all the same data types or an array that holds different data types. 

            let myMixedArray = ["apple", 10, true];
Let’s take a closer look at the syntax in the above array example:
  • The array is stored in a variable called myMixedArray
  • The array is represented by the square brackets [ ] and the content inside.
  • Each content item inside an array is called an element.
  • There are three different elements inside the array.
  • Each element inside the array is a different data type, which are: string, number and a boolean

Accessing Elements

Accessing elements of an array means retrieving specific values from our array. We do this using the index. Index tell us where exactly in the array the element is located. It's just like in your shopping list, you have items listed out with number from one to the last number on the list. However, numbering in JavaScript starts from Zero (0).

In our above example, JavaScript takes the 0 position, React takes the 1 Position while PHP takes the 2 position. We can then call an element using the index position.
