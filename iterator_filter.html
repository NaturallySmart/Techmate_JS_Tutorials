The .filter() Method

The .filter() method is a powerful array method that allows you to create new arrays by filtering the elements of an existing one based on specific criteria. It creates a new array with all elements that pass the test implemented by the provided function. If no elements pass the test, it returns an empty array. The returned result is not modified if the original array is sorted or has any side effects. The array is not mutated i.e it does not change.

Let's look at an illustration: 

                const numbers = [2, 5, 9, 4];
                let filteredNumbers = numbers.filter(function (number) {
                    return number % 2 === 0; // only even numbers will be included
                    });
                    console.log(filteredNumbers); // Output: [2, 4]
Let's look ay another example using strings: 
                const animalNames = ['Tiger', 'Lion', 'Dog', 'Antelope', 'Elephant'];
                const namesStartingWithA = animalNames.filter(function (name){
                    return name[0].toLowerCase() ==='a';
                    })
                    console.log(namesStartingWithA); // Output: ["Antelope"]
Another Example: 
                const names = ['Joy', 'Bella', 'Mike', 'Joe'];
                const shortNames = names.filter(name =>{
                    return name.length <=3;
                    });
                console.log(shortNames);//Output:['Joy', 'Joe']
In the last example;
  • We have an array called names which contains string elements
  • The filter method is used to create a new array shortNames with all elements that pass the test implemented by the provided function.
  • The arrow function in this case checks if each element of the array has length less than or equal to 3 characters.
  • The output will be an array containing only the elements from the original array whose length was less than or equal to 3.
