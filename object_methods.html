Object Methods

In JavaScript, objects can have their own methods which are functions that belong to the object and can be called on it. These methods can perform a variety of tasks such as calculating values or modifying properties.

When the data stored on an object is a function we call that a method. A property is what an object has, while a method is what an object does.

We are already familiar with Methods of some sort as we have been using it all along. For example: when you use the Math Object's Math.sqrt(), Math.max(), Math.min(), etc., these are all methods. Here, Math is global JavaScript Object while .sqrt(), .max() and .min() are methods on it. Console is also a global Object in JavaScript while .log() is a method on that object.

We can include methods in our object literals by creating ordinary, colon-separated key-value pairs. The key serves as our method’s name, while the value is an anonymous function expression (or any other type of function). This allows us to create custom methods for our objects.

Let's see how this works! 

                const car = {
                    mycarDesc: function() {
                        console.log('My car is a red Toyota Camry with a 
                        5 speed auto transmission. I bought the car from
                        a friend who was moving away');
                    }
                };
With the new method syntax introduced in ES6 we can omit the colon and the function keyword. So the above will look like this. 
                const car = {
                    mycarDesc() {
                        console.log('My car is a red Toyota Camry with a
                        5 speed auto transmission. I bought the car from
                        a friend who was moving away');
                    }
                };

Invoking Object Method

Object methods are invoked by appending the object’s name with the dot operator followed by the method name and parentheses: 

                car.mycarDesc(); // Prints:
                'My car is a red Toyota Camry with a 5 speed auto transmission. 
                I bought the car from a friend who was moving away'.
