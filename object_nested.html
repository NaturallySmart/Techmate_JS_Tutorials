Nested Objects

Most times, an object are often nested. That is, objects might have another object as a property which in turn could have a property that might be an array of other objects and so on. This can get pretty complex and it's important to know how to handle these situations.

In JavaScript, we use the dot notation (.) to access properties of nested objects. For example: 

                const person = {
                    firstName: "John",
                    lastName: "Doe",
                    age: 32,
                    address: {
                        streetAddress: "4567 Main St.",
                        city: "New York City",
                        state: "NY",
                        zipCode: "10001"
                        }
                        };
                console.log(person.address); // Output: {streetAddress: 
                "4567 Main St.", city: "New York City", state: "NY", zipCode: 
                "10001"}
