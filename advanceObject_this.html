Advance Objects | The this Keyword

In JavaScript, the "this" keyword is a special variable that refers to an object. It holds the reference of the current object and allows access to its properties and methods. When we use it inside a method or function, it represents the owner of that method or function.
See an Illustration below: 

                const car = {
                    Make: 'Toyota',
                    Model: 'Camry',
                    Year: 2019,
                    Color: 'Blue',
                    power: '203hp',
                    getCarDetails() {
                        return `This ${this.Make} ${this.Model} has a ${this.
                            power} and was made in ${this.Year}.`;
                        },
                };
                console.log(`${car.getCarDetails()}`)
