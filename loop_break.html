The break Keyword

The "break" keyword is used in a loop (for or while) to exit the loop prematurely when a certain condition is met. It stops executing the remaining code inside the loop and transfers control back to the statement following the loop. This is useful for breaking out of nested loops if a specific condition has been reached. 

            
                // Example: Break Out Of A Loop
                let i = 5;
                while(i > 0){
                    console.log("Looping...");
                    // If i equals 3, we will break out of the loop
                    if(i == 3){
                break;
                        }
                        i--;
                }
                console.log("Exited Loop!");

In this example, the "break" keyword is used within an if statement that checks whether the value of 'i' is equal to 3. When it does, the program breaks out of the loop by skipping over any further iterations. As a result, only three times "Looping..." would be printed before "Exited Loop!" was displayed.
