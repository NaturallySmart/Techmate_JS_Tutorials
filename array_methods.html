    • Array .length property

    Just as in Strings, arrays also have an in-built properties one of which is the .length property. It returns how many items are in the array.

    check below: 

                let arr = [5, 6, 7];
                console.log(arr.length); // Outputs: 3
    The above uses numbers, now let's try for strings: 
                let strArr = ["apple", "Sweet banana", "cherry"];
                console.log(strArr.length); // Outputs: 3

    Example 1

    The .push() Method

    This method is used to add an element to the end of an array. So let's say your array contains 3 elements and you want to add a fourth or more element to it, you use the .push method to do this.

    See example below: 

                let itemNumbers = ['item 0', 'item 1', 'item 2'];
                itemNumbers.push('item3', 'item4', 'item5');

            console.log(itemNumbers); //Outputs: ['item 0', 'item 1', 'item 2', 
            'item3', 'item4', 'item5'];
    Here is how it works:
  • We access the push method by using dot(.) notation, connecting push to itemNumbers with a period.
  • The push method is called like a function because it is a function, an in-built function that JavaScript allows us to use right on an array.
  • .push() can take a single argument or multiple arguments separated by commas. In this case, weâ€™re adding three elements: 'item 3' 'item 4' and 'item 5' to itemNumbers.
  • Notice that .push() changes, or mutates, itemNumbers. You might also see .push() referred to as a destructive array method since it changes the initial array.

    The .pop() Method

    Array .pop( ) method removes the last item in an array. 

                let itemNumbers = ['item 0', 'item 1', 'item 2'];
                const removed = itemNumbers.pop( );

            console.log(itemNumbers); //Outputs: ['item 0', 'item 1',]; 

            console.log(removed); // Outputs: item 2
    In the above example,
  • .pop() removed the the last item which is item 2
  • .pop() does not take any argument inside it's parenthesis, it simply just removes the last item in the array of itemNumbers.
  • .pop() returns the value of the last element. In the example, we store the returned value in a variable removed to be used for later.
  • When removed is called, it prints the element in the array that was removed
