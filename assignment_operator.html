The -= Operator

The Subtraction Assignment Operator subtracts a value from a variable.

Code Example: let x = 10; x -= 5; This will subract 5 from 10

Other types of Assignment Operators are:

  • The Multiplication (*=) Operator
  • The Exponential (**=) Operator
  • The Division (/=) Operator
  • The Remainder (%=) Operator


The Increment and Decrement Operator

Other mathematical assignment operators include the increment operator (++) and decrement operator (--).

The increment operator (++) is used to increase the value of a variable by one, while the decrement operator (--) is used to decrease the value of a variable by one.
