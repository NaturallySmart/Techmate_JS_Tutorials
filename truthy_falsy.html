Truthy and Falsy

As a developer, there are times you will want to check if a variable exist and if it has been assigned a value and not to necessarily equal a specific value. This will help you know if the datatype is empty, null or if it has a value. This is very important in development to debug errors in the code and make necessary corrections.

The list of falsy value includes:

  • An empty string " "
  • A variable assigned to a value of 0
  • A variable assigned to null meaning no value at all
  • Undefined which represent when a declared variable lacks a value
  • and NaN, or Not a Number, will return a false value when checked against a condition.
