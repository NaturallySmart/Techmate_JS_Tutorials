Accessing Object Properties

In Javascript there are two methods to access the property of an Object. One of such method is the Dot (.) Notation

The Dot . Notation

In JavaScript, we can access the properties of an object using dot notation. This is done by placing a period (.) between the variable holding the reference to the object and the property you want to access. For example: 

                let person = {
                    firstName: "John",
                    lastName: "Doe",
                    age: 35
                    };
                console.log(person.firstName); // Outputs: John
Does the above looks familiar? Remember how we usually find the length property of an string? 
                let str = "Techmate";
                console.log(str.length); // Outputs: 8
If we try to access a property that does not exist on that object, undefined will be returned. e.g: 
                console.log(person.favColor); // Outputs: Undefined
