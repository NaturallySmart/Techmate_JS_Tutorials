Introduction to Javascript

Data Types

Data types are the classifications we give to the different kinds of data that we use in programming. In JavaScript, there are seven fundamental data types:

  1. Number type: This is used for numerical values. It can have decimal or no decimal. For Example: 2, 5. 78, 5.78, 100 etc
  2. BigInt: Any number, greater than 2 53-1 or less than -(2 53-1), with n appended to the number: 1234567890123456n.
  3. String: Any grouping of characters on your keyboard (letters, numbers, spaces, symbols, etc.) surrounded by single quotes: ' ... ' or double quotes " ... ",. Some people like to think of string as a fancy word for text. for Example "Hello World", 'Hello' etc.
  4. boolean: true or false values. This is mostly used in statements or as an answer to a YES or No question.
  5. Null: This data type represents the intentional absence of a value, and is represented by the keyword null (without quotes).
  6. Objects: Collection of related information enclosed between curly braces {}.
  7. Undefined: This data type is denoted by the keyword undefined (without quotes). It also represents the absence of a value though it has a different use than null. undefined means that a given value does not exist.

They are the most basic data types in the language. Objects are more complex, and you’ll learn much more about them as you progress through JavaScript. At first, seven types may not seem like that many, but soon you’ll observe the world opens with possibilities once you start leveraging each one. As you learn more about objects, you’ll be able to create complex collections of data.
