Javascript Objects

In Javascript, Objects are used to store data in key-value pairs where each value can be of any type including other Objects or arrays. They provide a flexible way to organize and access the data. Objects permeates nearly every aspect of JavaScript programming.

in our previous lessons, we have already been exposed to using Objects, knowlingly or unknowlingly. With Objects we open our code to more complex possibilities. We can use JavaScript Objects to model real-world things, like a car, or we can use objects to build the data structures that make the web possible.
